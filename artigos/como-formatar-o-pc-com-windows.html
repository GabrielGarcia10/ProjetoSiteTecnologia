Como formatar o PC com Windows?

Veja o passo a passo de como formatar e instalar o Windows no computador


Formatar o PC consiste no processo de apagar todos os arquivos do computador, incluindo o sistema operacional, e instalá-lo novamente. Apesar de ser conhecido, o procedimento é um desafio considerável para o público menos entendido. Por isso, o Adrenaline resolveu montar um passo a passo de como formatar o computador, instalando o Windows ao final do processo.

Para uma formatação completa, será necessário contar com um pen drive de pelo menos 8 GB de espaço de armazenamento.

Importante: antes de iniciar a formatação, faça um backup! Se você possui algum documento ou arquivo importante, tanto no pen drive utilizado para formatar, quanto no PC que será zerado, salve estes arquivos em algum outro local. Bons locais para guardar estes arquivos são HDs externos, celulares, nuvem, ou mesmo outros computadores e pen drives.

Como formatar o PC?

A primeira parte necessária para a formatação é preparar os arquivos que executarão a limpeza e instalação do Windows através do pen drive. Para isso, basta acessar a página de download de software da Microsoft e escolher a versão que deseja instalar ao final do procedimento.

No nosso caso, optamos pelo Windows 11, mas boa parte do processo é similar para sistemas anteriores, já que o novo sistema pode não ser suportado computadores antigos. Na página, clique na opção "Criar mídia de instalação do Windows 11" e faça o download.

  1. Faça o download do arquivo da mídia de instalação do Windows

    2. Essa parte pode ser feita em qualquer computador. Não há necessidade de executá-la no PC que será formatado.

    Faça o download do arquivo da mídia de instalação do Windows
  2. Execute o arquivo e crie o pen drive bootável

    3. Em seguida, conecte o pen drive no computador e execute o arquivo "mediacreationtool" baixado. Aceite os termos de licença, mantenha as opções recomendadas para o computador e avance. Em seguida, escolha por "Unidade flash USB" e avance. Depois, escolha o pen drive utilizado avance e aguarde. Ao final, basta concluir.

    Execute o arquivo e crie o pen drive bootável
  3. No PC que será formatado, insira o Pen Drive com a máquina desligada e acesse o menu de boot

    4. A próxima etapa acontece no PC que será formatado. Com ele desligado, conecte o pen drive preparado. Depois disso, será necessário acessar o Menu de Boot da placa-mãe e este passo requer atenção. Para acessar o Menu de Boot, é necessário apertar um dos botões do teclado enquanto o PC é inicializado.

    O problema é que cada fabricante de placa-mãe opta por uma das teclas do teclado, por isso, é difícil indicar qual é a tecla para cada caso. Veja aqui um passo a passo de como saber qual é a placa-mãe do seu PC.

    Abaixo, listamos algumas opções, mas, mesmo dentro de uma mesma fabricante pode haver variação de acordo com cada modelo. Por isso, o ideal buscar por como acessar o Menu de Boot na sua placa-mãe específica.

    Como acessar a BIOS
    Como acessar a BIOS

    No caso do exemplo feito aqui, utilizamos um modelo da ASUS e pressionamos o botão F8 (seguidas vezes) enquanto o PC inicializava. Depois disso, já no Menu de Boot, escolhemos pelo Pen Drive entre as opções disponíveis.

    Isso faz com que o sistema leia os arquivos do dispositivo antes de ler o drive do PC. Assim, o boot ou inicialização acontecem por ele.

    Menu de boot
  4. Prossiga para a instalação do Windows

    5. A parte seguinte é de instalação Windows. Depois de iniciar a instalação, basta mudar a opção "teclado ou método de entrada" para ABNT 2 (ou manter padrão se for internacional) e avançar.

    Defina teclado ou método de entrada em ABNT2
  5. Insira a chave de ativação do Windows ou escolha "Não tenho a chave do produto"

    6. Se você possuir uma chave de ativação do Windows, pode inseri-la na página. Caso não tenha, basta escolher por "Não tenho a chave do produto".

    Chave de ativação do Windows

    O processo de comprar e adicionar a chave pode ser feito após o Windows ser instalado, acessando Configurações > Sistema > Ativação > Vá para a Loja. Se já possuir uma chave, basta seguir o caminho Configurações > Sistema > Ativação > Atualizar chave do produto > Alterar chave do produto.

  6. Escolha a instalação personalizada

    7. Em seguida, é o momento de optar pela versão do sistema operacional e avançar, aceitar os termos. Na página do tipo de instalação, opte pela opção "Personalizada: instalar apenas o Windows".

    Instalação personalizada do Windows
  7. Apague as partições do HD ou SSD do sistema e escolha o drive onde o Windows será instalado

    8. "Atenção nessa etapa, aqui acontece o processo que apaga definitivamente os arquivos do pc"

    A etapa seguinte dá ao usuário a opção de escolher onde deseja instalar o Windows. O sistema mostra as unidades de disco disponíveis no PC, isto é, os HDs ou SSDs disponíveis para instalação no sistema e suas respectivas partições. Quando um drive é instalado no Windows, o sistema operacional separa ele em algumas partes para o correto funcionamento.

    Neste momento, o que será feito é excluir as partições da Unidade para formatar todas as partes dele. Basta então, com um clique, selecionar uma partição por vez e clicar em excluir. Depois disso feito, sobrará apenas as partes "originais" dos HDs ou SSDs disponíveis do sistema (se houver dois conectados no PC, por exemplo, sobrarão duas opções, como no nosso caso, veja abaixo).

    Se houver dois drives (discos de armazenamentos), por exemplo, na máquina, escolha qual dos dois será usado para a instalação do Windows.

    No que não for utilizado para a instalação, selecione e clique na opção "novo" para tornar este drive como utilizável pelo Windows. Depois, no que for utilizado para a instalação, selecione avançar. Então, basta esperar a instalação do Windows.

    Escolher onde instalar o WIndows
  8. Configure o Windows como desejar

    9. Depois disso, resta configurar o Windows de acordo com as preferências de usuário. Uma dica: se não quiser vincular uma conta da Microsoft a este PC Windows, digite no espaço de e-mail test@test.com e como senha test. Uma mensagem de erro será exibida.

    Basta, então, avançar que o Windows dará a opção para criação de uma conta local. Espere as reinicializações, se necessário, e pronto! PC formatado.

    Depois da formatação, o ideal é prosseguir com as instalações e atualizações de drivers do sistema, como os de chipset, placa de vídeo e outros componentes.
